## 4 TÂCHES

### T1: Gemini (1-2h)
```
1. Lire: GEMINI_INTEGRATION.md
2. Obtenir clé API (Google AI Studio)
3. pip install google-generativeai
4. Copier: chatbot_service_GEMINI_TEMPLATE.py
5. Redémarrer backend
```

### T2: Modèles (1-2h)
```
1. Lire: MODELS_INTEGRATION.md
2. Créer: backend/app/ml/models/
3. Placer: naive_bayes_model.pkl
4. Placer: tfidf_vectorizer.pkl
5. Copier: classifier_TEMPLATE.py
6. Redémarrer backend
```

### T3: Dashboard (2-4h)
```
1. npm install recharts lucide-react
2. Copier: DashboardPage_TEMPLATE.jsx
3. Copier: DashboardPage.css
4. Redémarrer frontend
```

### T4: Tests (2-4h)
```
1. Lire: TESTING_GUIDE.md
2. pip install pytest
3. Créer: backend/tests/
4. pytest tests/ -v
```

## 🔑 COMMANDES ESSENTIELLES

```powershell

.\start_app.ps1

# Backend seul

npm run dev
```
